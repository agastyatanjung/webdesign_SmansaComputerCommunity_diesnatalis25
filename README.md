# webdesign_SmansaComputerCommunity_diesnatalis25
Ecoverse merupakan website yang kami kembangkan untuk meningkatkan kepedulian lingkungan melalui edukasi dan interaksi sederhana. Sebagai peserta lomba, kami berfokus menghadirkan platform yang informatif, mudah digunakan, dan dirancang dengan gaya modern untuk mengajak pengguna berperan aktif menjaga bumi

Cara Menjalankan Project

1. Clone repository:
git clone https://github.com/agastyatanjung/webdesign_SmansaComputerCommunity_diesnatalis25.git
2. Masuk ke folder project.
3. Buka file index.html di browser (Chrome/Firefox/Edge).

Atau akses versi online melalui GitHub Pages:
https://ecoversewebsite.netlify.app

Website siap dijalankan tanpa setup tambahan
